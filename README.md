# Jessie Doucette — Portfolio

This is the source code for my personal engineering portfolio, hosted at
[jessiedoucette.github.io](https://jessiedoucette.github.io).

## File structure

```
.
├── index.html       # All site content
├── style.css        # All styling
├── README.md        # This file
└── images/          # Profile photo + project images
    ├── profile.jpg
    ├── rx7-1.jpg
    ├── penguingineer-1.jpg
    └── mech3500-1.jpg
```

## Editing the site

- **Content** — open `index.html` in any text editor and edit the text directly. Headings, paragraphs, project bullets, and contact info are all in plain HTML.
- **Styling / colors** — open `style.css`. The color palette is defined at the very top under `:root`. Change those values to retheme the whole site at once.
- **Adding photos** — drop image files into the `images/` folder using the exact filenames listed above. The site automatically picks them up; if a file is missing, a placeholder appears instead.

## Image recommendations

| File                   | Used for          | Suggested size        |
| ---------------------- | ----------------- | --------------------- |
| `profile.jpg`          | About-me photo    | 800 × 1000 px (4:5)   |
| `rx7-1.jpg`            | RX-7 project hero | 1200 × 900 px (4:3)   |
| `penguingineer-1.jpg`  | Game project hero | 1200 × 900 px (4:3)   |
| `mech3500-1.jpg`       | LEGO car hero     | 1200 × 900 px (4:3)   |

JPEGs around 200–500 KB each are ideal — small enough to load fast, large enough to look sharp.
